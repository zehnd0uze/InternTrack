คู่มือการใช้งานระบบจัดการเวลาพิเศษ
(Secret Missing Logs Manager)

สำหรับ: ผู้ดูแลระบบ (Admin)
ระบบ: InternTrack

1. บทนำ (Introduction)

ระบบจัดการเวลาพิเศษเป็นฟีเจอร์ที่ถูกซ่อนไว้ (Secret Feature) สำหรับผู้ดูแลระบบ เพื่อใช้ในการเพิ่มหรือแก้ไขเวลาเข้า-ออกงานให้กับนักศึกษาในกรณีที่นักศึกษา "ลืมบันทึกเวลา" (Missing Logs) โดยระบบนี้จะทำการคำนวณชั่วโมงการทำงานและอัปเดตลงในฐานข้อมูลการเข้างาน (Attendance) และสมุดบันทึกประจำวัน (Daily Logs) โดยอัตโนมัติ

2. การเตรียมความพร้อมฐานข้อมูล (Database Setup)

⚠️ ข้อควรระวัง: หากยังไม่ได้รันคำสั่ง SQL เหล่านี้ใน Supabase ระบบจะไม่สามารถใช้งานได้

ผู้ดูแลระบบต้องนำไฟล์ Migration ไปรันใน Supabase SQL Editor ตามลำดับดังนี้:

  • รันไฟล์ 022_missing_attendance.sql เพื่อสร้างตารางสำหรับเก็บประวัติการลืมบันทึกเวลา
  • รันไฟล์ 023_add_time_to_missing.sql เพื่อเพิ่มคอลัมน์เวลาเข้า-ออกงานลงในตาราง

3. วิธีการเข้าถึงระบบ (How to Access)

เนื่องจากเป็นฟีเจอร์ลับ จะไม่มีเมนูแสดงในแถบด้านข้าง (Sidebar) วิธีการเข้าถึงมีดังนี้:

  1. เข้าสู่ระบบด้วยบัญชี ผู้ดูแลระบบ (Admin)
  2. ไปที่เมนู "จัดการข้อมูล" (Data Manager) จากแถบเมนูด้านซ้าย
  3. คลิกที่แท็บที่ 3 ซึ่งมีชื่อว่า "เวลาที่ขาดหาย" (ไอคอนรูปไฟล์มีเครื่องหมายตกใจ)
  4. ระบบจะแสดงหน้าต่างให้ใส่รหัสผ่าน ให้พิมพ์รหัสผ่าน: admin123
  5. คลิก "ยืนยัน" ระบบจะปลดล็อกและแสดงตารางจัดการเวลาพิเศษทันที (รหัสผ่านจะจำไว้จนกว่าจะออกจากระบบ)

4. วิธีการใช้งาน (How to Use)

4.1 การเพิ่มข้อมูลเวลาที่ขาดหาย (Add Missing Log)

  • คลิกปุ่ม "+ เพิ่มข้อมูลใหม่" มุมขวาบน
  • เลือกชื่อ นักศึกษา ที่ต้องการเพิ่มเวลา
  • ระบุ วันที่ ที่นักศึกษาลืมบันทึกเวลา
  • เลือก ประเภท (ลืมเข้างาน, ลืมออกงาน, หรือ ลืมทั้งเข้าและออกงาน)
  • ระบุ เวลาเข้างาน และ/หรือ เวลาออกงาน ตามความเป็นจริง
  • ใส่ หมายเหตุ (เช่น "ระบบมีปัญหา", "นักศึกษาลืมสแกน")
  • คลิก "บันทึก"

* เมื่อบันทึกสำเร็จ ระบบจะนำเวลาเข้า-ออกไปคำนวณชั่วโมงทำงานและอัปเดตในแท็บ "การเข้างาน" ทันที รวมถึงนำหมายเหตุไปใส่ใน "บันทึกประจำวัน" ของนักศึกษาคนนั้นในวันดังกล่าว

4.2 การแก้ไขข้อมูล (Edit Data)

  • ในตารางเวลาที่ขาดหาย ให้คลิกที่ไอคอน "ดินสอ" ด้านขวาสุดของรายการที่ต้องการแก้ไข
  • แก้ไขเวลาหรือหมายเหตุตามต้องการ แล้วกด "บันทึก" ระบบจะอัปเดตข้อมูลทั้งในหน้าต่างนี้และในตารางการเข้างานจริง

4.3 การลบข้อมูล (Delete Data)

  • คลิกที่ไอคอน "ถังขยะสีแดง" ด้านขวาสุดของรายการ
  • ระบบจะถามเพื่อยืนยันการลบ เมื่อกดยืนยัน ข้อมูลในประวัติการลืมบันทึกเวลาจะถูกลบออก (หมายเหตุ: การลบในหน้านี้จะไม่ได้ลบประวัติการเข้างานจริงในแท็บ "การเข้างาน" หากต้องการลบเวลาเข้างานจริง ต้องไปลบที่แท็บ "การเข้างาน" ด้วย)
